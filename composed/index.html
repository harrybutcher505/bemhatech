🍎 Mac users — read this first

macOS will block Composed the first time because it's not yet in the App Store. Here's how to open it:

  1. Right-click Composed.app → choose Open
  2. A dialog appears — click Open again to confirm

If you see "Composed cannot be opened" with no Open option:

  1. Go to System Settings → Privacy & Security
  2. Scroll down — you'll see "Composed was blocked"
  3. Click Allow Anyway, then try opening Composed again

This is a standard macOS step for apps outside the App Store. You only need to do it once.

How to install

1

Download for your platform

Click the Download button above for Mac or Windows. The file will land in your Downloads folder.

2

Install

Mac: Unzip → drag Composed.app to Applications → right-click and choose Open the first time.
Windows: Run Composed_setup.exe → click through the installer → launch from Start menu.

3

Get your Anthropic API key

Composed uses Claude AI. You'll need a free API key from console.anthropic.com — takes 2 minutes. Add it in Composed under Profile.

4

Upload your CV and start

Go to Profile → upload your CV (PDF, DOCX, or TXT). Then go to Outreach to find your 10 target companies, or Apply to draft a letter for a specific role.

📱 Desktop only — Composed is a Mac and Windows desktop application. It does not run in a browser. Your CV and API key stay on your own machine and are never uploaded to any server.
